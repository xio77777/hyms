# 项目简介

## 项目名称
HYMS（康复视神经训练系统）

## 项目目标
通过视觉追踪训练和情绪调节灯光，有效锻炼脑梗患者的视神经灵活性和专注力，防止感官退化。

## 目标用户
- 脑梗康复期患者
- 患者的照护人员

## 核心需求
1. 提供多感官刺激训练，帮助患者锻炼视神经灵活性
2. 具备情绪调节功能（舒缓/兴奋模式）
3. 操作极简，适合老年及康复期患者使用
4. 支持投屏电视，大屏优先设计

## 技术栈
- 前端：React + TypeScript + Vite
- 移动端：Capacitor（Android）
- 样式：Tailwind CSS
- 状态管理：Zustand

## 项目结构
- `/src` - 前端源代码
- `/android` - Android 原生代码
- `/public` - 静态资源
