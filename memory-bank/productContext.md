# 产品上下文

## 产品存在的原因
脑梗患者在康复期常常面临视神经功能退化和专注力下降的问题。传统的康复训练方式枯燥且难以坚持，需要一款能够在家中轻松使用、操作简单的视觉康复训练工具。

## 解决的问题
1. **视神经灵活性下降** - 通过视觉追踪训练，锻炼眼球追踪能力
2. **专注力不足** - 通过多感官刺激，提升注意力集中度
3. **情绪调节需求** - 提供舒缓和兴奋两种模式，帮助调节情绪状态
4. **操作门槛高** - 极简设计，大字体、大按钮，适合老年患者

## 产品工作方式
1. 用户打开应用后进入首页，展示所有训练模式入口
2. 点击"多感官刺激"进入训练界面
3. 可选择三种模式：
   - **视觉追踪训练**：移动的光点/光带吸引目光追随
   - **舒缓模式**：柔和缓慢变色的灯光效果，帮助放松
   - **兴奋模式**：色彩鲜明的快速变化灯光，改善精神状态
4. 可调节训练速度和亮度
5. 随时可返回首页

## 用户体验目标
- 沉浸式全屏体验
- 流畅的 60fps 动画效果
- 高对比度、大字体，确保可读性
- 操作简单，无需学习成本
- 支持遥控器/鼠标/触控多种操作方式
