## 当前状态
项目已完成所有核心训练功能（5大训练类型 + 呼吸训练 + 视力自测），支持多种投屏方式，具备完善的游戏化激励系统（成就、等级、打卡）、数据统计与分享功能，以及全面的用户体验优化（新手引导、护眼提醒、环境音效、键盘快捷键等）。产品已具备市场竞争力，可作为中国视觉康复训练领域的领先产品。

## 已知问题
- TypeScript 5.8.3 版本中 baseUrl 选项有弃用警告，不影响构建

## 项目决策演变
- 采用 Canvas + Renderer 模式处理复杂动画
- 使用 Zustand 替代 Redux 简化状态管理
- 使用 Capacitor 而非 React Native 进行移动端开发
- 亮度控制改用 CSS filter 优化渲染性能
- 添加音效系统提升用户体验
- 引入游戏化设计（成就、等级、经验值）提升用户粘性
- 增加呼吸训练模块，视觉+呼吸同步提升康复效果
- 增加视力自测功能，形成"检测-训练-追踪"闭环
- 护眼提醒系统，遵循20-20-20法则保护用户视力
- 全面支持键盘/遥控器操作，优化大屏投屏体验
